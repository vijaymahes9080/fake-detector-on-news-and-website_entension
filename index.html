Install & Load Developer Build

Run this extension locally in your Chrome browser by following these quick developer instructions.

1

Get Codebase

Ensure you have downloaded or cloned this repository to your local computer.

git clone https://github.com/vijaymahes9080/fake-detector-on-news-and-website_entension.git
2

Open Extensions

Launch Google Chrome and navigate to the Extensions management screen by typing this URL in your search bar:

chrome://extensions/
3

Enable Dev Mode

Locate the Developer Mode toggle switch in the top-right corner of the window and turn it ON.

4

Load Directory

Click the "Load unpacked" button in the top-left, and select the folder containing these files.
